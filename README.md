# ClickSport

## Установка

1. Установите зависимости для каждой части проекта:
   - В папке `/database`:
     ```bash
     npm install
     ```
   - В папке `/api`:
     ```bash
     npm install
     ```
   - В папке `/web`:
     ```bash
     npm install
     ```

## Запуск Backend

Перейдите в папку `/api` и выполните:

```bash
npm start
```

## Запуск Backend

Перейдите в папку `/web` и выполните:

```bash
npm start
```
